== Welcome to Treehaus

Treehaus is a tool to provide companies a way to boost their internal 
communication and training. It provides an easy way for company leaders to
post tasks and training requirements that any employee can do and gain
experience in the department of the desires. 


== Getting Started

1. Install Ruby, Postgres, Redis 

2. Start the server with ```rails s``` 

3. Start redis and the worker with ```foreman start``` 

4. Go to http://localhost:3000/
